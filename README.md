# Unemployment Analysis in India 📊

## Project Overview
This project focuses on analyzing unemployment trends in India using Python. 
The analysis includes data exploration, visualization, and basic time-series insights to understand unemployment patterns across different regions.

The project uses popular data science libraries such as Pandas, NumPy, Matplotlib, Seaborn, and Statsmodels.

## Objectives
- Analyze unemployment data in India
- Understand regional unemployment patterns
- Visualize unemployment trends using graphs
- Explore time-series patterns in unemployment rates

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels
- Jupyter Notebook

## Dataset
The dataset used in this project contains unemployment statistics in India including:
- Region
- Date
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)

## Project Workflow
1. Import required libraries
2. Load the unemployment dataset
3. Data preprocessing and cleaning
4. Exploratory Data Analysis (EDA)
5. Visualization of unemployment trends
6. Time-series decomposition to understand seasonal patterns

## Sample Analysis Performed
- Unemployment rate distribution
- Regional unemployment comparison
- Trend analysis over time
- Seasonal decomposition
